概要 (Introduction)
------
.NET Framework 4にチャートコントロールが増えたとのことなので使ってみました。
本来はデータベースを使った物になるのでしょうが、今回はサブディレクトリ容量を
円グラフとリストにして表示するサンプルです

環境 (Environment)
------
#### 開発環境  
Visual Studio 2010 / Visual Studio 2013

#### 確認環境  
Windows7 + .NET Framework 4



使い方 (Usage)
------
1. 起動  
起動するとドキュメントフォルダ配下にあるサブフォルダのディスク容量を解析し
円グラフとリストで表示します。

2. サブディレクトリを解析する  
右側に表示されているサブディレクトリリストから該当サブディレクトリを選択し、
「フォルダ解析」ボタンをクリックしてください。

3. 親フォルダに移動する  
サブディレクトリリストの上にある「ひとつ上へ」ボタンをクリックしてください。

4. 現在のフォルダを再解析する  
サブディレクトリリストの上にある「更新」ボタンをクリックしてください。
